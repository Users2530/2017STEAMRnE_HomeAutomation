## 각 파일의 설명

### Arduino 폴더

이 파일을 전부 다운로드한 다음, 아두이노에 업로드합니다.

[STEAMRnE_34C.ino](Arduino/STEAMRnE_34C.ino): 전자기기 제어 장치에 사용하는 코드입니다.
